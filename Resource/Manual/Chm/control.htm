2.1 Features

System

1. Nice and clean GUI(graphic user interface) plus user friendly.
2. Easy to configure and hassle free.
3. Smart resource and memory manager. Not resource hungry program !
4. Reliable and intelligent network protocol or layer.

General

1. 3 customer charge type, ability to swith between the type.
a) Pay As You Go.
b) Prepaid.
c) Fixed Time.
2. Complete statistic. Including daily, monthly, most popular Pc, most profitable day etc.
3. Sophisticated POS (point of sales) System.
4. Cafe Map, view or set your cafe computer layout.
5. Configurable item in point of sales system.
6. User can customize price scheme for different user.
7. Automatic monthly income calculation.

Pc Monitoring & Controlling

1. Lock & Unlock Pc by single Pc or silmutaneously.
2. Shutdown & Reboot Pc by single Pc or silmutaneously.
3. Chat or send ay message to user.
4. Sending tray ticker message.
5. Realtime monitor status.

Security

1. Username & password required to log in.
2. Multiple employee password & diffrent access to diffrent section.
3. Ability to logout and chage user without shutting down the server.

 

2.1 Configuration

On the first run, you will be prompt the configuration dialog, and detail of every
setting are as below.

Cybercafes Name - You caybercafe name.

E-mail - Your email, no email will be send to you for filling this.

Motto - Enter your cyercafe motto, give your cybercafe a spirit.

Pricing - Enter your normal pricing. The second input box is for the initial value, leave the value zero if do not to use the initial value. You can click
the button on the right of the pricing group.

Local Port - Listening port for communication between the server and the client. The
default port 56266. Change this value if you have problem that other program are
currently using this port.

Master Password - Your root password to access this system.

Monthly Overhead - Your monthly overhead including premise rental, electric/water
bils, others and worker salary.

Point Of Sales - This part shows item list and the price.

User Access - This part allow administrator to set their employee access
