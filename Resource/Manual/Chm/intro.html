1.1 About

Cafebonzer is a client-server application for controlling and managing a cybercafe. This system has been developed by an programming expertise on a Windows Platform. This systemwill give you total control or management of your Cybercafe or Pc Lab.

 

1.2 Requirement

CafeBonzer Controller :

Operating System Windows 95, 98, ME, 2000 (not tested)
Processor 200+ Mhz Pentium
Memory 16 Mb (minimum)
HardDisk 7 Mb

CafeBonzer Agent :

Operating System Windows 95(not recommended), 98, ME,2000(not tested)
Processor 200+ Mhz Pentium
Memory 16 Mb (minimum)
HardDisk 2 Mb

 

1.3 Installation

CafeBonzer Control (CbControl1.exe) must be install into a server from which
computer that will control all others terminal. Run the installer and follow the
instruction, the computer will restart afte installation. On the first run you will
be prompt with the configuration dialog, Username & Password are required to
complete the configuration.

CafeBonzer Agent (CbAgentR3.exe) is a client and must be install to every Pc
that you want to control and monitor. After the installation, it will prompt you
the configuration dialog, below is the most important step that you have to
follow to complete the installation :

  • Setting the server IP, the server is the computer that you have installed
    the CafeBonzer Control. Ex:- 192.168.1.100
  • The port must be same as the server port. Just leave the value for the
    default port(recommended)
  • Choose the apropriate password.
